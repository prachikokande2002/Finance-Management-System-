1)Project Overview

The Personal Finance Management System is a menu-driven, command-line application developed using Python to help users record, manage, and analyze their daily expenses efficiently. The system is designed using object-oriented programming (OOP) principles and provides reliable data persistence through CSV file handling.

This project enables users to track expenses by category, date, and description, generate detailed financial reports, and analyze spending patterns. It also includes backup and restore functionality to ensure data safety.

objective 
*To build a user-friendly expense tracking system

*To apply object-oriented design concepts in Python

*To implement file handling for persistent data storage

*To generate financial reports and spending analysis

*To ensure data reliability through backup and restore features
